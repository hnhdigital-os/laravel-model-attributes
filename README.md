## Requirements

* Laravel 5.5
* PHP 7.1

## Install

Via composer:

`$ composer require hnhdigital-os/laravel-model-attributes ~1.0`

## Configuration

Enable the model on any given model.

```php
use HnhDigital\ModelAttributes\Model;

class SomeModel extends Model
{

}
```
